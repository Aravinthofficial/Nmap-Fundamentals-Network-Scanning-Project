Nmap Fundamentals & Network Scanning Project

This project demonstrates my practical understanding of Nmap fundamentals and network scanning techniques in a safe virtual lab environment. I learned how to perform host discovery, port scanning, service enumeration, operating system detection, and basic vulnerability scanning using Nmap on Kali Linux.

Skills Learned:
• Nmap Fundamentals
• Network Scanning
• Port Enumeration
• Service Version Detection
• OS Detection
• TCP/UDP Scanning
• Basic Vulnerability Assessment
• Linux Commands
• Kali Linux
• Cybersecurity Fundamentals

Commands Practiced:
nmap <IP>
nmap -sV <IP>
nmap -O <IP>
nmap -A <IP>
nmap -sS <IP>
nmap -sU <IP>
nmap --script vuln <IP>

Project Objective:
To understand how cybersecurity professionals and SOC analysts use Nmap for network discovery, service identification, and basic reconnaissance in cybersecurity environments.

Tools Used:
• Kali Linux
• Nmap
• VirtualBox
• Ubuntu/Metasploitable Lab

Outcome:
Successfully learned Nmap fundamentals, identified open ports and running services, performed basic network enumeration, and improved practical cybersecurity skills through hands-on lab practice.# Nmap-Fundamentals-Network-Scanning-Project
